# 🎓 College Wallah

A simple and clean **College Landing Page Website** built while learning **HTML and CSS**.
This project helped me understand basic web development concepts like layout, navigation bar design, and styling with CSS.

---

## 🚀 Features

* 📌 Responsive navigation bar
* 🎓 College themed landing page
* 🎨 Clean UI design
* ⚡ Built using pure HTML & CSS
* 🌐 Font Awesome icons integration

---

## 🛠️ Tech Stack

* **HTML5**
* **CSS3**
* **Font Awesome**

---

## 📂 Project Structure

```
Project
│
├── index.html
├── index.css
└── assets (images/icons if added)
```

---

## 💻 How to Run the Project

1. Clone the repository

```
git clone https://github.com/ritesh01-hub/college-wallah-website.git
```

2. Open the folder

```
cd college-wallah-website
```

3. Open **index.html** in your browser.
